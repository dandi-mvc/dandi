  • Inherited from Disposable.remapDisposed

    • Defined in dandi/common/src/disposable.ts:236

After being called, Disposable.isDisposed will return true for target, and Disposable.getDisposedReason will return reason. target will also be "frozen" with Object.freeze, preventing further modification of the instance.
